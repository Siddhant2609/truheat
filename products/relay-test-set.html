Relay Test Set

Comprehensive testing of protective relays, over-current relays, earth fault relays, and circuit breakers with precision digital time measurement.

Request Quote → 📋 View Specs

Overview

These Relay Test Sets are ideal for testing protective relays, over-current relays, earth fault relays, and circuit breakers. Equipped with a precision digital time interval meter for accurate timing measurements.

Output Specifications

  • Current Output — 0–1/2/5/10 Amps (Open Circuit Voltage: 25V)
  • High Current — 0–50/100 Amps (Open Circuit Voltage: 5V)
  • AC Voltage Output — 0–250V, Capacity: 2 Amp
  • DC Voltage Output — 0–250V, Capacity: 1 Amp
  • Timer — 4-Digit Time Interval Meter (9.999 or 99.99 seconds)

Input

Standard input voltage: 240V AC, single phase.

Testing Applications

  • Over-Current Relays — Testing pickup, timing, and trip characteristics
  • Earth Fault Relays — Verification of earth fault protection settings
  • Circuit Breakers — Trip timing and coordination testing
  • Differential Relays — Testing differential protection schemes
  • Buchholz Relays — Gas accumulation relay testing
  • Directional Relays — Phase angle and directional testing
